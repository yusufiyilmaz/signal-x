# Katki Rehberi

## Gelistirme Ortami
```bash
git clone https://github.com/yusufiyilmaz/signal-x.git
cd signal-x
cargo build
cargo test
```

## Kod Standartlari

- `cargo fmt` ile formatlama
- `cargo clippy` ile lint kontrolu
- Her yeni ozellik icin unit test ekle

## Branch Yapisi

- `main` - stabil surum
- `midterm/isim-soyisim` - ogrenci branch
